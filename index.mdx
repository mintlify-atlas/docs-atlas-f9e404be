Password Pusher is an open source tool for sharing sensitive information safely. Create a one-time link that expires automatically after a set number of views or days — no more emailing passwords in plain text.

Use the [hosted service at pwpush.com](https://pwpush.com) or deploy your own instance with Docker in minutes.

## Push types

Password Pusher supports four types of secure pushes:

- **Text / password** — Share any secret text with a one-time link
- **File** — Upload and share files that self-delete after viewing
- **URL** — Share a URL that redirects once and then expires
- **QR code** — Generate a QR code from any text, shared securely

<Steps>
  <Step title="Create a push">
    Enter your secret, set expiration limits, and click **Push it!** to get a one-time link.
  </Step>
  <Step title="Share the link">
    Send the generated link to your recipient via email, chat, or any channel.
  </Step>
  <Step title="Link auto-expires">
    After the set number of views or days, the secret is permanently deleted. Nothing lingers.
  </Step>
</Steps>
